# 交互式shell脚本工具

## 安装

### 一键安装(推荐)
> 推荐 root 用户

```sh
wget -N  http://raw.githubusercontent.com/sshpc/s/main/s.sh && chmod +x s.sh && sudo ./s.sh
```

> 再次执行只需要输入 “s” 

```sh
 root@server:~#  s
```
